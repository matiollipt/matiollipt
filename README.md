## Current Work

**Founder & Lead Data Scientist — [AidBio](https://aidbio-ai.github.io/)** *(2024 – present)*

> Modular scientific software for Agriculture 3.0, molecular biology, computational biology, and applied R&D.

- **AidWeather** — Makes NASA POWER climate data analysis-ready with validated retrieval, caching, and pre-processing
- **AidViz** — Turns NASA POWER, agroclimatic, and farm datasets into clear visual analytics
- **AidFarm** — Makes agricultural interpretation and reporting more scalable with LLM-enabled visual and text reasoning
- **AidGSEA** — Makes public GEO RNA-seq studies reusable through metadata validation, differential expression, and pathway analysis
- **Aid2GO** — Improves protein-function prediction by combining biological networks, sequence features, text embeddings, and Gene Ontology labels
- **gPPIpred** — Delivers published graph-ML protein-protein interaction prediction with interpretable residue-level inference; DOI: [10.17912/micropub.biology.001796](https://doi.org/10.17912/micropub.biology.001796); code: [Hugging Face](https://huggingface.co/datasets/1143Joana/gPPIpred_code/tree/main/gPPIpred/)
- **SynViz** — Explores synesthetic sound-to-color and sound-to-touch visualization with real-time audio analysis
